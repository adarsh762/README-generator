01 How should this be generated? 🎛️

Kept in your browser's memory only for this tab — never saved to disk, never sent anywhere except directly to the provider you picked. Don't have one? Get a key here.

Uses the site owner's shared key with a small daily limit — nothing for you to sign up for or paste in. If it says the daily limit's been reached, try the Claude or Groq modes instead.

Template mode builds your README from patterns in what you paste — no AI call, no signup, completely free. It detects a package.json if present, and turns lines starting with -, *, •, or ✅ into a features list. It does not read the "Additional instructions" field below, and won't write original prose the way the Claude/Groq modes do — for a genuinely detailed, written-for-your-project README, use one of those instead.
